# Sovereign — Landing Page

Landing page for **Sovereign**, the agentic Telegram client.

**Live site:** [www.tgsovereign.com](https://www.tgsovereign.com)

**Core repo:** [github.com/tgsovereign/core](https://github.com/tgsovereign/core)

## Development

```bash
npm install
npm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.

## Tech Stack

- [Next.js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com)
- [shadcn/ui](https://ui.shadcn.com)
